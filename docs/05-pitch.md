# Estratégia de Pitch

## 1. O Problema (30 seg)
Foco na dificuldade de compreensão de termos financeiros e na falta de personalização no ensino de finanças pessoais para o brasileiro médio.

## 2. A Solução (1 min)
Apresentação do Edu como um agente de IA que utiliza RAG (Geração Aumentada de Recuperação) para conectar conceitos teóricos aos dados reais de transações e perfil do usuário.

## 3. Demonstração (1 min)
Apresentação visual da interface Streamlit, demonstrando:
- Análise dinâmica de gastos (transacoes.csv).
- Sugestões baseadas no perfil (perfil_investidor.json).
- Explicação didática de termos complexos.

## 4. Diferencial (30 seg)
Destaque para o uso de IA garantindo privacidade de dados e a capacidade de manter o contexto histórico para um atendimento contínuo.
